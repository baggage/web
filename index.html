baggage.io

A simple and free web-to-email api

Getting started

subscribe

You will receive an email with your subscription tokens. 
$ curl -is https://api.baggage.io/subscribe/you@domain.com

HTTP/1.1 200 OK
Content-Type: application/json;charset=utf-8
X-Content-Type-Options: nosniff
Content-Length: 31

{"message":"subscription sent"}

From: Baggage.io <no-reply@baggage.io>
To: you@domain.com
Subject: New Baggage.io subscription

Your new subscription:

id:              e189f55f616b6156227511c83756f54c
email token:     d7073a8e087ec3e3c500c4eeee3abc841754ef8ba05d5d22c20103070046e6d4
admin token:     5ad4addbfd944a53dfd51eb52ee5fbfaa59b9031020bddfab2e4e254b4c0c0c2

rotate tokens:   http://api.baggage.io/rotate/e189f55f616b6156227511c83756f54c?token=5ad4addbfd944a53dfd51eb52ee5fbfaa59b9031020bddfab2e4e254b4c0c0c2
unsubscribe:     http://api.baggage.io/unsubscribe/e189f55f616b6156227511c83756f54c?token=5ad4addbfd944a53dfd51eb52ee5fbfaa59b9031020bddfab2e4e254b4c0c0c2

Your subscription will expire after 7 days of inactivity.

send a message:

http://api.baggage.io/send/e189f55f616b6156227511c83756f54c?token=d7073a8e087ec3e3c500c4eeee3abc841754ef8ba05d5d22c20103070046e6d4&subject=hello&body=world

Regards,
baggage.io

The id is unique for your subscription, and you can have as many subscriptions per email address as you like.

Now send an email

send

Use the id and email token 
$ curl -is 'https://api.baggage.io/send/e189f55f616b6156227511c83756f54c?token=d7073a8e087ec3e3c500c4eeee3abc841754ef8ba05d5d22c20103070046e6d4&subject=hello&body=world'

HTTP/1.1 200 OK
Content-Type: application/json;charset=utf-8
X-Content-Type-Options: nosniff
Content-Length: 18

{"message":"sent"}

From: no-reply@baggage.io
To: you@domain.com
Subject: hello

world

You can also use a POST request.

Rotate the tokens

rotate

Use the id and admin token to change the tokens at any time 
$  curl -is 'https://api.baggage.io/rotate/e189f55f616b6156227511c83756f54c?token=5ad4addbfd944a53dfd51eb52ee5fbfaa59b9031020bddfab2e4e254b4c0c0c2'

HTTP/1.1 200 OK
Content-Type: application/json;charset=utf-8
X-Content-Type-Options: nosniff
Content-Length: 21

{"message":"rotated"}

To: you@domain.com

The id stays the same.

Unsubscribe

unsubscribe

Use the id and admin token to unsubscribe 
$ curl -is 'http://localhost:4567/unsubscribe/b39f6b8d46be132c84c79b5da811e540?token=2c3f59cc7209bd44d8519d1b990bfa0fdfc801eb1a25468f1d4afd42dd023728'

HTTP/1.1 200 OK
Content-Type: application/json;charset=utf-8
X-Content-Type-Options: nosniff
Content-Length: 26

{"message":"unsubscribed"}

From: Baggage.io <no-reply@baggage.io>
To: you@domain.com
Subject: Baggage.io b39f6b8d46be132c84c79b5da811e540 unsubscribed

Hi,

b39f6b8d46be132c84c79b5da811e540 has been unsubscribed. Thank you.

Regards,
baggage.io

The subscription will also expire after 7 days inactivity (configurable).
